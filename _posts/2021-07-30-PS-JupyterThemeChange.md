# jupyter 테마 변경

1. cmd 실행
2. juptyerthemes 설치
```python
pip install jupytertheme
```
3. 테마 명령어
    * 테마 리스트
      ```python
      jt -l
      ```
    * 테마 변경
      ```python
      jt -t 테마 이름
      ```
    * 테마 초기화
      ```python
      jt -r
      ```
